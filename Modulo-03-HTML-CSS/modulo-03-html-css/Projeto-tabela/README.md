# Projeto Tabela HTML

Projeto desenvolvido para praticar a criação de tabelas em HTML e estilização com CSS.

## Tecnologias utilizadas

- HTML5
- CSS3

## Estrutura do projeto

📄 Tabela.html  
📁 css/

## Objetivo

Praticar:
- Criação de tabelas HTML
- Uso de `table`, `thead`, `tbody` e `tfoot`
- Organização de dados em linhas e colunas
- Estilização com CSS

## Como visualizar

Abra o arquivo `Tabela.html` no navegador.

```bash
xdg-open Tabela.html
```
